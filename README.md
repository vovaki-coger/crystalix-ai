# Crystalix AI — ИИ-аналитик дуэлей CSH

Прозрачный оверлей поверх Minecraft, который анализирует дуэли в Custom Steve House на Crystalix.

## Что умеет

- Захватывает твой экран в реальном времени
- Отправляет скриншот в Ollama (локальный ИИ с vision)
- Показывает вердикт: кто победит, шанс победы, уровень угрозы, тактику
- Работает полностью офлайн, без серверов
- Горячая клавиша F9 для быстрого анализа
- Прозрачное окно поверх игры (перетаскивается мышью)

## Установка

### 1. Установи Python
Скачай с [python.org](https://python.org), при установке отметь **"Add Python to PATH"**

### 2. Установи Ollama
Скачай с [ollama.ai](https://ollama.ai) и установи

### 3. Скачай модель llava (умеет видеть картинки)
```
ollama pull llava
```
> Вес ~4-8 ГБ. Можно попробовать `llava:7b` для скорости или `llava:13b` для точности

### 4. Установи зависимости
Запусти `install.bat` (двойной клик)

### 5. Запуск
Запусти `start.bat` (двойной клик)

## Использование

1. Запусти Ollama (должна работать в фоне)
2. Запусти `start.bat`
3. Зайди в Minecraft → Crystalix → CSH → встань в очередь на дуэль
4. Когда видишь снаряжение противника — нажми **F9** или кнопку в оверлее
5. Через 10-60 секунд получишь вердикт

## Обучение ИИ

Открой файл `knowledge.md` в любом редакторе (Блокнот, VS Code) и добавь:
- Описание классов
- Характеристики снаряжения
- Тактики
- Любую инфу об игре

ИИ автоматически учитывает всё что написано в этом файле при каждом анализе.

## Настройка

Открой `config.json`:
- `model` — модель Ollama (`llava`, `llava:13b`, `llava:7b`)
- `hotkey` — горячая клавиша (по умолчанию F9)
- `timeout_seconds` — максимальное время ожидания ответа
- `monitor` — номер монитора (1 = основной, 2 = второй)

## Скорость ответа

Зависит от твоего железа:
- С GPU (NVIDIA): 10-25 секунд
- Без GPU (CPU): 30-90 секунд

Для ускорения: используй `llava:7b` вместо `llava`

## Проблемы

**"Ollama не запущена"** — открой Ollama (иконка в трее) или запусти `ollama serve` в терминале

**"Нет модели llava"** — выполни `ollama pull llava` в терминале

**Чёрный экран на скриншоте** — игра в полноэкранном режиме. Переключись на оконный или оконный без рамки
